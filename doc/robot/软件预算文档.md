# 预算清单（按软件详细设计 SD 条目对齐）

> 说明：
> - 金额单位：人民币（元），为可执行的参考预算（标准方案），用于立项/采购沟通。
> - 人力成本按 `40 万/人·年` 计入；折算 `≈ 33333 元/人月`（40万÷12，取整）。
> - 表内“可选项”默认不计入总价，除非明确采购/纳入本期范围。
> - 系统硬件与集成预算不在本文内，见 `doc/robot/系统预算文档.md`。
> - 人月时间按 `doc/robot/软件详细设计文档.md` 的 SD-16.1 为准（合计 **8.0 人月**）。

## 4. 汇总（标准方案）

> 说明：总价（标准方案）= 人力成本（按 SD-16） + 非人力（按需） + 预备费（建议按“非人力”的 10% 计入，便于采购沟通）。

- 人力成本（8.0 人月）：266664 元
- 非人力小计（本期，按需）：19000 元（示例：Token 12000 + 预录音外采 5000 + 样例 2000）
- 预备费（按非人力 10%）：1900 元
- 总价（标准方案，示例）：**287564 元**

> 备注：若“非人力按需项”全部为 0（完全本地/内部制作），则总预算可按 **266664 元** 计；如需分期实施，请按第 2 节的人月拆分与第 3 节的可选项单独立项。
