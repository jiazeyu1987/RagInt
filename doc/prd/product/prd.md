# 最终版产品需求文档（V2/V3）

## 1. 产品定位
本产品是一套 **可 7x24 稳定运行的展厅级智能语音讲解与问答系统**，支持自然打断、低延迟响应，并可基于展厅知识库与复杂文档进行精准回答。

目标用户：
- 展厅 / 博物馆 / 科技馆 / 企业展厅
- 需要长期无人值守、可插话、可扩展的语音讲解系统

## 2. 核心目标
- 观众可自然提问、随时打断
- 系统使用指定音色进行讲解
- 首字延迟低、交互自然
- 长时间稳定运行
- 网络异常不断场

## 3. 核心能力需求

### 3.1 语音交互能力
- 多麦克风阵列（4/6 麦）输入
- 支持声源定位（DOA），锁定正前方观众
- 支持回声消除（AEC），播报时不误触发
- 支持自然语言打断（Barge-in）

### 3.2 对话与问答能力
- 支持连续多轮问答
- 高置信问题快速响应
- 不确定内容可明确拒答或引导

### 3.3 知识与文档能力
- 展厅结构化知识库
- 支持上传 PDF 文档并进行问答
- 能回答文档中的细节问题（文本/表格）

### 3.4 性能与稳定性
- 首字延迟 < 1.5 秒
- 打断成功率 > 95%
- 7x24 小时稳定运行
- 支持断网/接口异常自动降级

## 4. 用户交互流程
1. 系统待机（呼吸态）
2. 前方观众讲话触发唤醒
3. 实时识别并生成回答
4. 语音播报（可被打断）
5. 观众插话 → 立即停止 → 新一轮对话

## 5. 非功能性需求
- 可远程运维与监控
- 日志可追溯
- 可模块化升级
- 支持后续数字人/屏幕形象接入

## 6. 验收标准
- 连续运行 72 小时无崩溃
- 现场插话立即生效
- 高频问题响应 < 0.3 秒
- 复杂文档问答准确率满足业务要求
