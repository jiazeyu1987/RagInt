# Demo版需求文档（V1）

## 1. 背景与目标
本 Demo 用于 **3–5 天内完成可演示原型**，在展厅场景中展示“用指定声音进行问答讲解”的能力。  
**核心原则：**
- 技术架构与最终产品保持一致
- Demo 演示后只做“能力替换与增强”，不推倒重来

## 2. Demo目标
- 支持观众语音提问 → 语音回答
- 使用指定音色（GPT-SoVITS）
- 支持前端按钮控制“开始 / 停止 / 打断”
- 延迟可接受、全流程稳定、不崩溃

## 3. 功能范围（Scope）

### 3.1 必须功能（Must Have）
- 基础语音问答（单轮）
- 前端控制是否开启问答
- 固定知识问答（20 条，100% 准确）
- RAG + LLM 生成补充回答
- 分段 TTS 播放（模拟流式）
- 手动打断（按钮触发）

### 3.2 非目标（Out of Scope）
- 真正硬件级 AEC / DOA
- 多人同时对话
- 多语言支持
- 复杂 PDF / 表格解析

## 4. 用户交互流程
1. 前端点击【开始问答】
2. 系统进入监听状态
3. 用户提问
4. 系统识别 → 生成 → 播放语音回答
5. 播放期间可点击【打断】
6. 前端点击【停止问答】返回待机

## 5. 成功标准（Demo 验收）
- 问“介绍一下这个展厅”能正确回答并播音
- 前端可控制启停与打断
- 从用户说完话到开始出声 ≤ 3 秒
- 全流程无崩溃，可连续运行 ≥ 30 分钟
